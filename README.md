# Search and sort algorithms using C
Compiled to webassembly with emscripten

## Prerequisites
- [emscripten](https://emscripten.org/docs/getting_started/downloads.html)
- [cmake](https://cmake.org/download/)

## Commands

To run app:

```bash
make build && make run
```
